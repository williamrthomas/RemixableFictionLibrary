Import from Project Gutenberg

Admin Import Tool

This is an administrative tool for directly importing books from Project Gutenberg. All imported books must be in the US public domain (generally published before 1929).

Project Gutenberg Import Form

Enter the numeric ID of the book on Project Gutenberg, e.g., "1342" for Pride and Prejudice

How to Find the Book ID

  1. Visit Project Gutenberg
  2. Find the book you want to import
  3. Click on the book to go to its detail page
  4. Look at the URL in your browser's address bar
  5. The URL will look like: https://www.gutenberg.org/ebooks/1342
  6. The number after "/ebooks/" is the Book ID: 1342
Important Notes
  • Only import books that are in the US public domain (generally published before 1929)
  • Project Gutenberg branding will be automatically removed during import
  • The system will attempt to download EPUB format first, then fall back to text format if EPUB is not available
